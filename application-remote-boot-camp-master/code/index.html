The simplest explanation of backend is what the user does not see or interact with in their browser, it stays on the server. Backend is like the web page's brain, if you as a user enter an address in the browser, it is not in your own computer. It is found elsewhere on the web and is called a backend, it can consist of one or more servers that the browser contacts and asks to have a page sent to it. It is therefore often called backend code server-side code. Most of the logic is usually on the backend side. There are many languages ​​to choose from on the server side such as Java, Python and Javascript
